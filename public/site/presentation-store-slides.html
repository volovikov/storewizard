Что такое storewizard?

Хранилище

Обратите внимание на панель Фильтр. Используя ее вы можете гибко настраивать представление в таблице. Можно выбрать по дате, можно по категории и т.д.

Поля формы меняются в зависимости от структуры данных вкладки. Если есть поля типа Дата, то в панели фильтр также будут поле для выбора промежутка времени.

Вы также можете отобрать для показа несколько записей, перечислив их идентификаторы в поле id через запятую, например: 1,10,110
